# 🫀 Heart Pulse Monitoring & Beat Counter System

A comprehensive hardware and Proteus-simulated heart pulse monitoring system designed as part of the **ELE402** curriculum. This project utilizes analog operational amplifiers for precise biometric signal conditioning, a buzzer for real-time auditory pulse indication, and a digital display component to track and count total heartbeats.

---

## 📁 Repository Structure

Based on the repository root layout:

* **`ELE402_Circuit_Analysis.docx`** – Detailed project report, circuit equations, and technical documentation.
* **`ELE402_Circuit_Analysis.pptx`** – Presentation slides summarizing the project design, challenges, and results.
* **`ProteusSimulation.DSN`** – Main Proteus circuit design file for running live simulations.
* **`ProjectDemo.mp4`** – A video demonstration showing the physical hardware operation and simulation run.
* **`LICENSE`** – Project licensing terms.
* **`README.md`** – This documentation file.
